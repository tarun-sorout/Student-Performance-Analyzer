# Student Performance Analyzer

This is a simple data analysis project built using Python and Pandas. The aim of this project is to analyze student marks and generate useful insights from the dataset.

## Dataset

The dataset contains 10,000 student records with marks in different subjects:

- Math
- Science
- English
- Computer
- Social Studies

Other information like Student ID, Gender, and Class is also included.

## What I Did

- Loaded the dataset using Pandas
- Calculated total and average marks
- Found the top 10 performers
- Identified highest scores in each subject
- Assigned grades based on average marks
- Generated summary statistics
- Performed class-wise and gender-wise analysis
